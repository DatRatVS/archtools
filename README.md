## tui.sh

Have a TUI to select from a list of scripts to run. This is meant to be run from the Arch ISO, and will pull the latest version of the script from GitHub.
- **Intended to be run from the Arch ISO.**
- **DO NOT RUN THIS IN YOUR SYSTEM.**

```bash
curl -sL https://is.gd/uapVBb | bash

# Or..

curl -sL https://raw.githubusercontent.com/DatRatVS/archtools/main/tui.sh | bash
```
